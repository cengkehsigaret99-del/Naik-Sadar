# QA Bahasa Modul Kebersihan

Gunakan checklist ini sebelum perubahan UI dianggap selesai.

## Checklist

- Judul halaman memakai istilah umum.
- Tombol memakai kata kerja yang jelas.
- Deskripsi layar tidak menilai pengguna.
- Instruksi dapat dilakukan dalam waktu singkat.
- Istilah subjektif diganti menjadi istilah kondisi atau hasil.
- Kata puitis hanya dipakai jika tidak mengganggu kejelasan.
- Bahasa tetap cocok untuk rumah, gudang, admin, dan produksi.
- Tidak ada klaim berlebihan tentang kondisi batin pengguna.

## Lulus QA jika

- Pengguna memahami apa yang harus dilakukan.
- Teks tidak terasa menghakimi.
- Teks tetap selaras dengan karakter Naik Sadar.
- Teks bisa dipakai untuk konteks pribadi dan kerja.

## Prioritas Revisi

1. Tombol
2. Judul layar
3. Deskripsi layar
4. Pilihan jawaban
5. Catatan riwayat
