# HTML Copy Patch Cleanliness

Dokumen ini berisi panduan penggantian teks untuk halaman Modul Kebersihan.

## Target File

- kebersihan-sadar.html
- kebersihan-5s.html

## Prinsip Patch

- Ubah teks UI saja.
- Jangan ubah CSS jika tidak diperlukan.
- Jangan ubah alur JavaScript jika tidak diperlukan.
- Pertahankan localStorage yang sudah berjalan.

## Patch kebersihan-sadar.html

Gunakan istilah:
- Modul Kebersihan
- Observasi
- Cek Awal
- Jeda Napas
- Tindakan
- Evaluasi Sesudah
- Catatan

Ganti pilihan kondisi menjadi:
- Terkendali
- Cukup rapi
- Perlu dirapikan
- Mengganggu fokus
- Sering tertunda

Ganti pilihan hasil menjadi:
- Area lebih rapi
- Area lebih mudah digunakan
- Akses lebih lancar
- Masih perlu dilanjutkan
- Cukup untuk hari ini
- Perlu evaluasi ulang

## Patch kebersihan-5s.html

Gunakan nama:
Mode 5S

Ganti Rawat menjadi Pelihara.

Gunakan kalimat:
5S membantu membuat standar area yang sederhana dan mudah diperiksa.

Ganti Jejak menjadi Catatan.

## Catatan Bahasa

Bahasa UI harus menjelaskan tindakan dan hasil. Hindari istilah yang terlalu personal, puitis, atau menilai pengguna.
