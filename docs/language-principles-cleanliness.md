# Language Principles Cleanliness

Prinsip bahasa Modul Kebersihan:

1. Instruksi layar harus umum dan objektif.
2. Bahasa pengguna menjelaskan tindakan, bukan kondisi batin.
3. Gunakan kata kerja yang dapat dilakukan.
4. Hindari kalimat yang terasa menilai pengguna.
5. Rujukan filosofis dipakai sebagai kerangka desain internal, bukan bahasa utama UI.

Istilah utama:
- Modul Kebersihan
- Observasi
- Jeda Napas
- Tindakan
- Evaluasi
- Catatan
- Hasil
- Standar Area
- Pemeliharaan Area
