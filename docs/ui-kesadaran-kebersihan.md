# UI Kesadaran Kebersihan

## Tujuan UI

UI harus sederhana, bersih, dan mudah dipakai.

## Prinsip

- Satu layar satu fokus.
- Tombol utama harus jelas.
- Teks pendek.
- Warna lembut.
- Hindari tampilan ramai.

## Komponen

### Dashboard
- Judul modul.
- Ringkasan sesi.
- Tombol mulai.

### Pilih Area
- Card area.
- Nama area.
- Status terakhir.

### Napas
- Instruksi singkat.
- Timer sederhana.
- Tombol selesai.

### Tindakan
- Tiga pilihan tindakan kecil.
- Estimasi waktu.

### Refleksi
- Skor sesudah.
- Rasa sesudah.
- Catatan singkat.

## Microcopy

- Mulai dari satu bagian kecil.
- Tidak perlu sempurna hari ini.
- Lihat dulu, napas dulu, baru bergerak.
- Satu langkah cukup untuk hari ini.

## Dashboard Metric

- Total sesi.
- Area paling sering dipilih.
- Rata-rata skor sebelum.
- Rata-rata skor sesudah.
- Sesi terakhir.
