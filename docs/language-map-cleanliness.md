# Language Map Cleanliness

Peta istilah untuk UI Modul Kebersihan.

- rasa: kondisi atau hasil
- jejak: catatan
- panik: perlu jeda
- menghindar: sering tertunda
- tegang: mengganggu fokus
- refleksi: evaluasi
- rawat: pelihara
- sadar: evaluatif atau terukur

Contoh:
- Evaluasi kondisi area sebelum memulai.
- Pilih satu tindakan yang bisa dilakukan sekarang.
- Ambil jeda singkat sebelum melakukan tindakan.
- Catat hasil setelah tindakan selesai.
