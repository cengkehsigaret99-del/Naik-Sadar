# UI Copy Standard

Standar bahasa UI untuk Modul Kebersihan.

Istilah utama:
- Modul Kebersihan
- Observasi
- Jeda Napas
- Tindakan
- Evaluasi
- Catatan
- Hasil
- Standar Area
- Pemeliharaan Area

Microcopy:
- Mulai dari satu area kecil.
- Pilih satu tindakan yang bisa dilakukan sekarang.
- Ambil jeda singkat sebelum memulai.
- Catat hasil setelah tindakan.
